# labredes-deploy_automatizado_provisionamento
Projeto desenvolvido para a disciplina de Laboratório de Redes. Este repositório contém os códigos de Infraestrutura como Código (IaC) e Gerência de Configuração responsáveis pelo provisionamento, estruturação de redes e implantação de serviços de uma plataforma distribuída operando em ambiente de nuvem OpenStack.
